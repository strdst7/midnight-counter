# Screenshots

Place your challenge screenshots here using these exact filenames so they render
in the main [`README.md`](../README.md) automatically:

| Filename                 | What to capture                                             |
|--------------------------|-------------------------------------------------------------|
| `compile-output.png`     | Terminal output of `npm run compile`                        |
| `contract-address.png`   | The `Contract Address: …` deploy line (or explorer view)    |
| `tests-passing.png`      | Green result of `npm test` (optional)                       |

Tip: on macOS, `Cmd+Shift+4` captures a region to your Desktop; move the file
here and rename it to match.
